# Criando o FE com REACT e Next.js

## Pré-requisitos

- Node: '>=20.9.0'

## Instalando a versão 22

```bash
nvm install 22 
```

## Instalando dependências

```bash
npm install    
```

## Instalando o React

```bash
npx create-next-app@latest bank
```

`Success!` Created bank at /home/rfahham/projetos/bank/FE/bank


## Executando

```bash
npm run dev
```
